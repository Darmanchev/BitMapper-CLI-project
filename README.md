# 🛠 BitMapper-CLI: Multi-Format Image Processor

**BitMapper-CLI** — это консольный графический редактор на C++, поддерживающий семейство форматов **Netpbm (PBM, PGM, PPM)**. Проект реализует полноценную иерархию классов для обработки монохромных, серых и полноцветных изображений.


* **Поддержка всех типов Netpbm:**
    * **PBM (P1):** Черно-белые изображения (битовые карты).
    * **PGM (P2):** Изображения в оттенках серого (Grayscale).
    * **PPM (P3):** Полноцветные изображения (RGB).
* **Глубокий полиморфизм:** Единый интерфейс для работы с разными форматами через базовый класс `Image`.
* **Расширенный функционал:** Теперь команды `grayscale` и `monochrome` реально преобразуют цветные изображения (PPM -> PGM -> PBM).

## 🚀 Основные возможности

* **Session Management:** Работа с несколькими файлами одновременно.
* **Трансформации:** `negative`, `rotate` (90° влево/вправо), `flip` (горизонталь/вертикаль).
* **Smart Paste:** Вставка одного изображения в другое с автоматической подстройкой размера холста.
* **Undo/Redo:** Полная история действий для каждой сессии.

## 🏗 Архитектура классов

Проект построен на принципах SOLID:
* `Image` (Base): Абстрактный интерфейс.
* `PBMImage`, `PGMImage`, `PPMImage`: Конкретные реализации логики форматов.
* `Session`: Контейнер для истории правок и текущего состояния изображения.
* `SessionManager`: Оркестратор всех открытых сессий.
